# KidWorld - Educational Web Platform for Children

## Overview
KidWorld is a web-based educational platform for children to explore national symbols such as flags, flowers, animals, and landmarks.

## Technologies
- Frontend: HTML, CSS, JavaScript, Bootstrap
- Backend: Node.js, Express.js
- Database: Oracle SQL Developer

## How to Run
1. Clone this repo
2. Setup Oracle DB and run SQL scripts in /sql/
3. Start backend using: `node index.js`
4. Access frontend via browser

## Project Structure
- /sql: SQL schema and data inserts
- /routes: API endpoints
- /FE: Static frontend files

## Report
This repository supports the report for the KidWorld project.
